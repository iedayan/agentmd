# AgentMD Skills

Agent skills for Cursor, Codex, Claude Code, and Gemini CLI.

## agentmd

Execute, validate, and score AGENTS.md files via the AgentMD CLI.

**Install (Cursor):**
```bash
cp -r skills/agentmd ~/.cursor/skills/
```

**Install (Codex):**
```bash
cp -r skills/agentmd ~/.agents/skills/
```

**Install (project-local):** Copy into `.cursor/skills/agentmd` or `.agents/skills/agentmd` in your repo.

## Submitting to awesome-agent-skills

When the skill has community usage, submit a PR to [VoltAgent/awesome-agent-skills](https://github.com/VoltAgent/awesome-agent-skills):

- Add under "Community Skills" → "Development and Testing" or "Other"
- Format: `**[agentmd/agentmd](https://github.com/agentmd/agentmd/tree/main/skills/agentmd)** - Execute, validate, and score AGENTS.md via CLI`
