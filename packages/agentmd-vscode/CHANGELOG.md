# Changelog

All notable changes to the **AgentMD** extension will be documented in this file.

## [0.1.0] - 2026-03-02

### Added
- **Initial Release** of the AgentMD VS Code extension.
- **Real-time Validation:** Live diagnostics for `AGENTS.md` files, highlighting missing build/test blocks and syntax errors.
- **Agent-Readiness Scoring:** Visual readiness badge (0-100) in the status bar with actionable feedback.
- **Template Library:** Support for generating optimized `AGENTS.md` files for 17+ frameworks (React, Next.js, FastAPI, Go, etc.).
- **Dry-Run Preview:** command preview for safely inspecting workflows directly from the editor.
- **Language Support:** Custom highlighting and configuration for the `AGENTS.md` specification.
