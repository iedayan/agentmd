Contributing

- Prerequisites: Node 20, PNPM
- Install: pnpm install
- Build: pnpm -r run build
- Test: pnpm -r run test
- Lint: pnpm run lint
- Commit: follow conventional-like messages; keep PRs focused
- PRs: ensure tests pass locally, describe changes, and run lint

## Recommended skills for development

For Cursor/Codex users, install these skills for better AGENTS.md workflows:

- **agentmd** (this repo): `cp -r skills/agentmd ~/.cursor/skills/` or `~/.agents/skills/`
- **getsentry/agents-md**: Create and manage AGENTS.md. See docs → Ecosystem.
- **vercel-labs/next-best-practices**, **openai/security-best-practices**: Browse [awesome-agent-skills](https://github.com/VoltAgent/awesome-agent-skills).
